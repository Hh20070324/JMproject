# JM 漫画下载器 v2 — 用户指南

---

## 📦 环境准备（仅首次）

### 第 1 步：安装 Python

> 如果你已有 Python 3.7+，跳过此步。

1. 打开 https://www.python.org/downloads/
2. 下载最新版 Python 安装包
3. 运行安装包，**务必勾选底部的 `Add python.exe to PATH`**
4. 点击 Install Now，等待完成

验证：按 `Win + R`，输入 `cmd` 回车，输入 `python --version`，显示版本号即成功。

### 第 2 步：安装依赖

1. 双击 `setup.bat`
2. 等待安装完成（约 1-2 分钟，需要联网）
3. 看到「安装完成」后按任意键关闭

### 第 3 步：开启代理（建议）

禁漫天堂对部分 IP 有地区限制。如果你在用 Clash Verge Rev 等代理工具，确保代理已开启即可，程序会自动走系统代理。

## 🚀 下载漫画

1. 双击 `一键下载.bat`
2. 浏览器会自动打开页面（如未打开，手动访问 `http://127.0.0.1:58080`）
3. 在输入框中输入禁漫车号（纯数字），按**回车**或点击「添加」
4. 每输入一个车号即可继续添加下一个，最多同时下载 2 个
5. 观察实时进度条，完成后显示「✅ PDF 已生成」

## 📁 文件输出

| 文件夹 | 内容 | 结构 |
|--------|------|------|
| `Pictures/` | 下载的原始图片 | `Pictures/车号/章节名/00001.jpg` |
| `PDFs/` | 自动打包的 PDF | `PDFs/车号.pdf` |

两个文件夹在首次下载时自动创建，无需手动操作。

## ⚙️ 高级配置

用记事本打开 `option.yml` 可修改以下参数：

```yaml
download:
  threading:
    batch_count: 16   # 同时下载的图片数（越大越快，越耗带宽）
  image:
    suffix: .jpg      # 图片格式：.jpg / .png
  retry_times: 3      # 失败自动重试次数
```

## ❓ 常见问题

### Q: 双击 bat 闪退？
在该文件夹空白处按 `Shift + 右键` →「在此处打开 PowerShell」，输入 `python server.py` 回车，查看具体报错。

### Q: 提示 ModuleNotFoundError？
依赖未安装。重新双击 `setup.bat`。

### Q: 下载报错 Restricted Access / 一直等待？
禁漫限制了你当前的 IP。开启代理后重试。

### Q: 关闭浏览器后任务还在吗？
在。服务器在后台继续运行，重新打开 `http://127.0.0.1:58080` 即可查看进度。

### Q: 怎么获取车号？
在禁漫网站上浏览本子时，网址里的数字就是车号。例如：
`https://18comic.vip/album/1236513/` → 车号 = `1236513`

### Q: 复制到其他电脑能用吗？
可以。把整个 `JM-Download&wrap_program` 文件夹复制过去，在新电脑上运行一次 `setup.bat` 安装依赖即可。

### Q: 下载速度太慢？
修改 `option.yml` 中的 `batch_count` 值（比如改为 32），并确保代理网络通畅。
