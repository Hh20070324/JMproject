# JM 漫画下载器 v2.1 — 用户指南
by TYZQ

---

## 📦 发行版用户

解压 `JM-Downloader-v2.1.0-Windows-x64.zip` 后双击 `JM-Downloader.exe`。发行版已经内置 Python 和运行依赖，无需执行环境安装。

请整体移动程序文件夹，不要只移动 EXE；`_internal` 文件夹是程序运行所必需的。

如果正式版无法启动，可以运行 `JM-Downloader-Debug.exe` 查看终端错误；两个 EXE 使用相同的配置、图片和 PDF 目录。

---

## 🛠️ 源码版环境准备（仅开发者）

### 第 1 步：启动程序

双击 `start.bat`。首次运行时，它会自动调用 `scripts/setup.ps1` 配置独立的 `.venv` 并安装固定版本依赖；以后启动不会重复安装。

### 第 2 步：开启代理（建议）

禁漫天堂对部分 IP 有地区限制。如果你在用 Clash Verge Rev 等代理工具，确保代理已开启即可，程序会自动走系统代理。

---

## 🚀 下载漫画

1. 发行版双击 `JM-Downloader.exe`；源码版双击 `start.bat`
2. 等待独立桌面窗口打开
3. 在输入框中输入禁漫车号（纯数字），按**回车**或点击「开始下载」
4. 每输入一个车号即可继续添加下一个，最多同时下载 2 个
5. 观察实时进度条，等待任务状态变为「已完成」

### 界面说明

- 「下载任务」显示首图、标题、实时进度、失败原因和重试操作。
- 「漫画库」支持按车号搜索，并可筛选有图片或有 PDF 的项目。
- 「打开图片」使用资源管理器；「查看 PDF」使用系统默认 PDF 阅读器。
- 右上角「实时更新正常」表示桌面界面已连接到本地任务服务。

## 📁 文件输出

| 文件夹 | 内容 | 结构 |
|--------|------|------|
| `Pictures/` | 下载的原始图片 | `Pictures/车号/章节名/00001.jpg` |
| `PDFs/` | 自动打包的 PDF | `PDFs/车号.pdf` |

两个文件夹在首次下载时自动创建，无需手动操作。

桌面窗口中的「漫画库」会直接扫描这两个文件夹。可以分别打开图片、查看 PDF、重新生成 PDF，或只删除其中一种文件。正在下载的漫画会暂时禁止删除和重新生成。

## ⚙️ 高级配置

用记事本打开 `option.yml` 可修改以下参数：

```yaml
download:
  threading:
    batch_count: 16   # 同时下载的图片数（越大越快，越耗带宽）
  image:
    suffix: .jpg      # 图片格式：.jpg / .png
  retry_times: 3      # 失败自动重试次数
```

## ❓ 常见问题

### Q: 正式版双击后没有窗口？
运行 `JM-Downloader-Debug.exe` 查看终端错误，同时检查 `logs/app.log`。

### Q: 源码版 start.bat 闪退？
在项目目录打开 PowerShell，运行 `.\.venv\Scripts\python.exe desktop.py`。需要浏览器调试时再运行 `.\.venv\Scripts\python.exe app.py`。

### Q: 提示 ModuleNotFoundError？
虚拟环境中的依赖不完整。删除 `.venv` 后重新双击 `start.bat` 修复。

### Q: 下载报错 Restricted Access / 一直等待？
禁漫限制了你当前的 IP。开启代理后重试。

### Q: 关闭桌面窗口后任务还在吗？
有下载任务时，程序会先显示原生确认对话框。确认关闭会终止进行中的任务并退出程序；取消则继续下载。

### Q: 怎么获取车号？
在禁漫网站上浏览本子时，网址里的数字就是车号。例如：
`https://18comic.vip/album/1236513/` → 车号 = `1236513`

### Q: 复制到其他电脑能用吗？
发行版可以。请整体复制解压后的 `JM-Downloader` 文件夹，不要遗漏 `_internal`。源码版不要复制 `.venv`，应在目标电脑重新运行 `start.bat`。

### Q: 下载速度太慢？
修改 `option.yml` 中的 `batch_count` 值（比如改为 32），并确保代理网络通畅。

## 🔧 项目结构

```
JM-Download&wrap_program/
├── desktop.py            ← 桌面窗口入口
├── app.py                ← 浏览器调试入口
├── jm_downloader/        ← API、任务调度、下载、PDF 及漫画库模块
├── server.py             ← 旧启动方式兼容入口
├── download_worker.py    ← 旧导入方式兼容模块
├── jpg2pdf.py            ← PDF 命令行兼容入口
├── option.yml            ← 下载配置
├── requirements.txt      ← 固定版本依赖清单
├── static/
│   ├── index.html        ← Web 前端结构
│   ├── app.css           ← 界面样式
│   └── app.js            ← 前端交互
├── scripts/
│   └── setup.ps1         ← 环境配置实现
├── python installer/     ← 内置 Python 安装程序
├── .venv/                ← 本机环境（自动生成，不参与分发）
├── start.bat             ← 唯一双击入口
└── README.md             ← 技术文档
```
