# JM 漫画下载器 —— 使用指南

## 这是什么？

一个**禁漫天堂漫画下载工具**。输入车号 → 自动下载所有章节图片 → 自动打包成 PDF。

全程只需要**双击 + 输入数字**，不需要敲任何命令。

---

## 第一次使用（只需做一次）

### 第 1 步：安装 Python

如果你电脑上已经装过 Python，跳过这一步。

1. 打开 https://www.python.org/downloads/
2. 点击黄色大按钮下载最新版
3. 运行下载的安装包
4. **务必勾选底部的 `Add python.exe to PATH`**（这一步非常重要！）
5. 点击 Install Now，等待安装完成

> 验证方法：按 `Win + R`，输入 `cmd` 回车，在黑色窗口里输入 `python --version` 回车。如果显示 `Python 3.x.x` 就说明装好了。

### 第 2 步：安装依赖包

1. 双击 `setup.bat`
2. 等待自动安装完成（需要联网，约 1-2 分钟）
3. 看到「安装完成」提示后按任意键关闭

> 如果提示「未检测到 Python」，说明第 1 步没做好。重新安装 Python 并确认勾选了 `Add python.exe to PATH`。

### 第 3 步：开启代理（可选但建议）

禁漫对部分 IP 地区有限制。如果你已经在用 Clash Verge Rev 等代理工具，确保代理已开启即可，程序会自动走系统代理。

没有代理也可能成功，取决于你的网络环境。

## 每次使用：下载漫画

### 方法一：GUI 窗口（推荐）

1. 双击 `一键下载.bat`
2. 在弹出的窗口中输入禁漫车号（纯数字，例如 `1236513`）
3. 点击「开始下载」或按回车
4. 等待日志显示「全部完成」
5. PDF 文件生成在项目文件夹下，名为 `车号.pdf`

### 方法二：命令行（适合批量处理）

按 `Win + R`，输入 `cmd` 回车，然后：

```bash
cd /d "这个文件夹的路径"
python JM.py 1236513
```

车号换成你要下载的即可。不写车号默认下载 `1236513`。

## 可选功能：手动打包图片

如果你已经有了一堆散图（比如从别处下载的漫画图片），想单独合并成 PDF：

1. 把图片复制到 `打包专用` 文件夹里
2. 双击 `手动打包.bat`
3. PDF 生成在项目文件夹下，名为 `打包专用.pdf`

> 图片会按文件名中的数字自然排序（`00001` → `00002` → `00010`），不会出现 `00001 → 00010 → 00002` 的错乱。

## 文件结构说明

```
JM-Download&wrap_program/
├── setup.bat            ← 首次使用：双击安装依赖
├── 一键下载.bat          ← 日常使用：双击启动下载 GUI
├── JM_gui.py            ← GUI 主程序（由 bat 调用）
├── JM.py                ← 命令行版本
├── jpg2pdf.py           ← 图片打包 PDF 引擎
├── option.yml           ← 下载配置（线程数、图片格式等）
├── 手动打包.bat          ← 手动把打包专用的图片合成 PDF
├── 打包专用/             ← 手动打包模式的图片入口
└── README.md            ← 本说明文件
```

下载后的漫画会出现在这个文件夹下，结构如下：

```
1236513/                 ← 以车号命名的文件夹
├── 第1話 章节标题/       ← 章节子文件夹
│   ├── 00001.jpg
│   ├── 00002.jpg
│   └── ...
└── ...
1236513.pdf              ← 自动生成的完整 PDF
```

## 常见问题

**Q: 双击 bat 闪退？**
A: 在 bat 文件上右键 → 编辑，确认里面的内容没有被杀毒软件拦截。也可以在该文件夹空白处按 `Shift + 右键` → 「在此处打开 PowerShell 窗口」，输入 `python JM_gui.py` 回车，看报错信息。

**Q: 下载报错 "Restricted Access"？**
A: 你的 IP 被禁漫限制了。开启代理后重试。

**Q: 下载报错类似 ModuleNotFoundError: No module named 'xxxx'？**
A: 依赖没装好。重新双击 `setup.bat` 安装。

**Q: 打包 PDF 时报错？**
A: 确认 `setup.bat` 已成功运行。如果出错信息包含 "PIL"，说明 Pillow 没装上，重新运行 `setup.bat`。

**Q: 想改下载速度/图片格式？**
A: 用记事本打开 `option.yml`，修改里面的配置：
- `batch_count: 16` → 数字越大下载越快（也越耗带宽）
- `suffix: .jpg` → 改为 `.png` 则输出 PNG 格式

**Q: 能把项目放到别的电脑上用吗？**
A: 可以。把整个 `JM-Download&wrap_program` 文件夹打包成 ZIP，复制到别的电脑上解压，然后从「第一次使用」的第 1 步开始做即可。所有路径都是自动适配的，放在任何位置都能运行。

**Q: 怎么获取禁漫车号？**
A: 在禁漫网站上浏览本子时，网址里的数字就是车号。例如 `https://18comic.vip/album/1236513/` 的车号就是 `1236513`。
